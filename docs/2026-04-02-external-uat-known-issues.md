# my-agent 外部测试：已知问题清单 (2026-04-02)

各位测试者，在开始测试前，请知悉以下已知局限。如果你在测试中遇到这些情况，属于预期范围内，请在记录中注明即可，不必纠结于此。

## 1. 稳定性与鲁棒性
- **索引损坏风险**：Archive 索引曾出现过单行格式错误导致读取失败的情况。虽然已加入容错和自动重建机制，但在极端频繁操作下，仍可能存在不稳定性。
- **并发限制**：目前系统主要设计为单任务单路径执行，不支持高并发任务或异步队列。

## 2. 输出与信息密度
- **噪音较高**：单次运行（Run）会生成大量文件（Archive contract 写入多文件），信息噪音可能较大，字段理解难度需验证。
- **界面缺失**：目前仅支持 CLI 命令行界面，暂无 Web UI 或图形化诊断工具。

## 3. 环境与依赖
- **Python 环境**：建议在干净的 Python 环境下运行。虽然我们验证过安装指令，但不同操作系统的依赖冲突仍可能导致首步失败。
- **网络依赖**：任务运行（Run）过程可能依赖外部模型 API，请确保你的网络环境可以正常访问相关服务。

## 4. 功能缺失（此版本暂不提供）
- 不支持自动重新规划（Adaptive Replanning）。
- 不支持数据库持久化状态（目前为文件系统驱动）。
- 不支持全文搜索或复杂的 DSL 查询。
