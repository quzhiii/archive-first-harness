# User Study Moderator Script

## Intro

今天我们会给你两个 agent failure debugging 小任务。
你的目标是：尽快判断“这次失败的主要原因是什么”。

## Condition A: Raw Logs

请阅读提供的原始日志材料，回答：
1. 失败发生在哪一步？
2. 你认为根因是什么？

记录：开始时间、结束时间、答案。

## Condition B: Archive Compare

请阅读提供的 archive compare 输出，回答同样两个问题：
1. 失败发生在哪一步？
2. 你认为根因是什么？

记录：开始时间、结束时间、答案。

## Post-task Questions

1. 哪种材料更容易帮助你定位问题？为什么？
2. compare 输出里最有帮助的一项是什么？
3. 哪一项最 confusing？
